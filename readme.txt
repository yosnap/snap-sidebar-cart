== Descripción ==

Snap Sidebar Cart es un plugin para WooCommerce que agrega un carrito lateral que se muestra automáticamente cuando los usuarios agregan productos al carrito o cuando hacen clic en selectores CSS específicos.

**Características principales:**

* Carrito lateral que se muestra automáticamente al agregar productos o al hacer clic en selectores configurables
* Visualización de productos relacionados en un slider configurable
* Opciones de personalización de estilos para adaptarse a cualquier tema
* Animaciones suaves al agregar o eliminar productos
* Los productos se eliminan del carrito cuando la cantidad llega a 0
* Muestra el precio del envío y el subtotal con IVA incluido
* El título muestra "Carrito de compra ([número-de-items-en-el-carrito])"
* Preloader durante las operaciones de AJAX
* Al hacer hover sobre productos relacionados, muestra una imagen de la galería

== Instalación ==

1. Sube la carpeta `snap-sidebar-cart` al directorio `/wp-content/plugins/`
2. Activa el plugin a través del menú 'Plugins' en WordPress
3. Configura las opciones del plugin en Ajustes > Snap Sidebar Cart

== Preguntas frecuentes ==

= ¿Es compatible con cualquier tema de WordPress? =

Sí, el plugin es compatible con cualquier tema que utilice WooCommerce correctamente.

= ¿Puedo personalizar los colores y estilos? =

Sí, el plugin tiene opciones de personalización en el panel de administración.

= ¿Cómo puedo mostrar el botón del carrito en mi tema? =

Puedes usar el shortcode `[snap_sidebar_cart_button]` o agregar el selector CSS de tu botón en la configuración del plugin.
